# TODO Backend (Node.js + Express + TypeScript)

간단한 TODO 백엔드를 위한 Node.js 기본 구조입니다.

## 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

기본 포트는 `4000`이며, 브라우저에서 `http://localhost:4000` 접속 시

```json
{ "message": "TODO backend is running 🚀" }
```

응답을 확인할 수 있습니다.

## 빌드 및 프로덕션 실행

```bash
npm run build
npm start
```
